Tek Tonik

Tek Tonik is a fictional, technology based e-commerce site built using Laravel on the back end and VueJs on the front. The visitor can browse a range of products, create an account and checkout using the Stripe api. They will then receive an email notification of their purchase and a dashboard is provided to keep track of orders in addition to other user information. The database used was mysql and no real credit card information is exchanged or persisted.

Booklist App

This booklist app was built entirely from vanilla JavaScript and it's data is stored and updated locally without need for an external database. Although simple now, it will grow to include a backend made entirely in pure php and the UI will be updated to reflect these changes. It was conceived to demonstrate work outside of frameworks.

Client Panel

A simple react ledger application that manages a user's financial arrangements. Balances and information can be updated and a registration system is in place for security. The database used is Google's Firebase and the CSS framework Bootstrap was used for the styling.
